# Example JSON References

These example files are JSONC-style references for human editing and LLM-assisted object generation.

- Use them as field/enum/unit guides.
- Remove `//` comments before feeding them to strict JSON parsers.
- The simulator normalizer still accepts additional fields, but these examples reflect the intended v2.4/v2.5 authoring shape.

Included examples:

- `Blue_Defended_Asset_Template.jsonc`
- `Radar_Node_Template.jsonc`
- `Kinetic_Gun_Template.jsonc`
- `Jammer_Node_Template.jsonc`
- `Red_OWA_Drone_Template.jsonc`
- `Terrain_Object.jsonc`
- `Scenario_Instance.jsonc`
- `Full_Template_Skeleton.jsonc`
