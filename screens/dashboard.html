Workflow Summary

The active scenario is edited live in Scenario Builder. Generator drafts only replace the live scenario when you explicitly click Generate From Wizard.
Template duplication is blocked by fresh generator builds and deterministic template-reference handling.
Environment Extractor and Template Manager remain standalone utilities in external_util/, with an in-app reference module for workflow handoff.
