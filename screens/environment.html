Environment Extractor

The extractor remains a standalone utility so terrain extraction, DEM handling, and scenario packaging can evolve without destabilizing the simulator shell. Use the buttons below to launch the external tools, then import the generated scenario JSON back into the simulator.

Recommended Workflow

  • 1. Extract terrainUse the standalone extractor for imagery, DEM, local-relief terrain segmentation, and scenario packaging.
  • 2. Review templatesUse Template Manager for bulk template authoring and reference JSON generation.
  • 3. Import and refineLoad the generated scenario here, then use Scenario Editor for live object and terrain debugging.

Extractor Notes

Current extractor terrain detection uses local relief, not map-wide average height, to emphasize radar-relevant spikes and anomalies.
The standalone utility still handles CORS fallbacks, contour guard rails, and scenario-ready JSON packaging outside the simulator shell.
