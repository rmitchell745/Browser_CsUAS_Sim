Primary Workflow
Edit draft scenario, stage it, run it, then review reports.

Recommended Workflow

  • 1. Quick StartLoad a built-in demo or blank scenario from Demo & Tutorial.
  • 2. Edit DraftUse Template Editor and Scenario Editor to change the draft scenario in memory.
  • 3. Stage DraftUse Stage Current Scenario to promote the draft into the active scenario used by Run.
  • 4. ExecuteRun single scenarios or Monte Carlo batches from Run Scenario.
  • 5. DebriefUse View Reports for summaries, timelines, and export-ready outputs.

Companion Utilities

Standalone helpers stay separate from the simulator shell so they can evolve without destabilizing the main UI.

Environment Extractor packages map imagery, terrain, and scenario-ready JSON.
Template Manager supports standalone template review and bulk authoring workflows.
